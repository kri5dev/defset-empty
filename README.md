Simple static page with ASCII art, ready for Vercel.

## Update ASCII

Replace the contents of `ascii/defset.txt` with your ASCII art.

## Local preview

```bash
npx serve .
```

(or open `index.html` directly in a browser.)

## Deploy to Vercel

- Import the repo in the Vercel dashboard or run:

```bash
npm i -g vercel
vercel --prod
```

Framework preset: "Other". Output directory: root.
